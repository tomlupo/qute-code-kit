# Frontend Fix

When: the frontend looks terrible and needs a quick visual overhaul.

## Prompt

This frontend looks terrible. Fix it:

- Instead of emojis, use icons
- Fix padding so every component is spaced perfectly — not too close, not too dispersed
- Goal: sleek, premium, minimalist — like a spa in Switzerland
- Design to match what a working professional would pay thousands for, in a way that would make Steve Jobs smile
- Pick a cohesive color palette and stick to it — avoid unnecessary colors
- Ensure responsive and elegant on both desktop and mobile
