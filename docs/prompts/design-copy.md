# Design Copy

When: you see a website/app design you like and want to replicate its visual style.

## Steps

1. Take a screenshot of the design you love
2. Drop it into Claude/ChatGPT with the prompt below
3. Use the output as a design system for your app

## Prompt

Create a design.json that captures the style of this layout. Include structure, spacing, fonts, and colors so I can use it as a design system for my app. The goal is to help AI replicate this exact look.

## Usage

Then tell your coding agent: "Use this design.json for styling only."
