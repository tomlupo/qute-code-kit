# Daily 15-Minute Protocol

**0 prep. Same routine every day.**

## 1) Speak (5 min)
Talk out loud about:
- your day / plans
- sport & training
- travel memory / next trip
- a tech tool you use
- markets / investing habits

**Rule:** speed > correctness.

## 2) AI correction (5 min)
Use `prompts/correct.md`.

## 3) Anki phrases (5 min)
Add the extracted phrases to Anki, or review from the provided deck.
