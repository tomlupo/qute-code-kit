# Italian Reactivation System (A2/B1 → B2)

**Time budget:** 15 min/day (Mon–Fri) + 1 tutor session (Thu)  
**Goal:** restore fast, automatic speaking + rebuild “glue” basics (prepositions, pronouns, connectors).

## Daily loop (15 minutes)
1. **Speak (5 min)** — out loud, no prep
2. **AI correction (5 min)** — paste transcript
3. **Anki (5 min)** — learn extracted phrases

## Weekly
- **Wed:** topic arrives → AI generates ideas (3–5 min)
- **Thu:** 15-min prep + tutor conversation
- Optional passive input (podcast/YouTube) whenever

## Files you’ll use most
- `prompts/correct.md`
- `prompts/tutor_prep.md`
- `anki/italian_phrases_all_600.tsv`
- `anki/by_tag/*.tsv`
